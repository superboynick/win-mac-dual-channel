# AirJet 数字复原计划：先尽可能像产品，再谈改进

版本：R0.1  
日期：2026-07-12  
当前目标：**没有 AirJet 实物、CT、拆解图或内部测量数据的条件下，基于公开专利、官方规格和教程，构建一个在结构、工作机理、封装尺寸和系统性能量级上尽可能接近 AirJet Mini 类产品的仿真模型。**

> 这不是“优化微射流散热器”的项目。此阶段只回答一个问题：我们能否构建一个可信的、可复现的 AirJet 数字样机（digital surrogate）？任何改变孔径、材料、阵列和控制策略的设计创新，全部放到复原模型完成以后。

## 1. 成功标准

“尽可能像”不能只指外观。R0 复原要同时满足四层相似性：

| 层级 | 必须相似的内容 | R0 完成判据 |
|---|---|---|
| 封装 | 外形长宽厚、进/排气方向、热源相对位置 | 与 AirJet Mini 官方规格在可获得维度上相容 |
| 架构 | 顶板进气、中央锚定压电执行片、顶/底腔、孔板、冲击间隙、侧向排气 | CAD 和剖面图逐项对应公开专利 |
| 动力学 | 超声频率、膜片模态、腔体声学共振、周期吸入/喷出和流量整流 | 结构谐响应与瞬态 CFD 形成自洽的周期解 |
| 系统级 | 热移除、输入功、背压、噪声/流速量级 | 在官方公开工作点附近量级相容；不能只匹配一个指标 |

没有实物，不能写“真实 AirJet 的完整数字孪生”。正确名称是：**public-evidence-constrained AirJet reconstruction**（公开证据约束的 AirJet 复原模型）。

## 2. 当前最高可信结构版本：R0-A

R0-A 选用公开专利中最完整、并与 AirJet 工作机理相符的“中央锚定”实施例作为几何母版：

```text
外界空气
   ↓  顶板中央/局部进气口
┌──────────────────────────┐
│ 顶板 + 顶腔（声学腔）      │
├──────────────────────────┤
│ 压电复合执行片            │  ← 中央锚定；两侧/周边悬臂振动
├──────────────────────────┤
│ 底腔（小间隙、流动整流）  │
├──────────────────────────┤
│ 多孔孔板                  │  ← 垂直微喷口
├──────────────────────────┤
│ 冲击间隙 / 热扩散面       │  ← 射流撞击后径向排出
└──────────────────────────┘
        → 侧向排气道 → 设备外部/回流路径
```

关键点：它不是传统风扇，也不是简单的“上下都吸吐”的合成射流。公开专利描述的目标是：上行程时尽量从顶腔补气而非从喷孔倒吸；下行程时空气穿过孔板高速冲向受热面，从而在一个周期内产生净向下流量。

## 3. 参数账本：什么可以直接用，什么必须标定

唯一参数来源表在 `evidence/airjet_reconstruction_ledger.csv`。任何写入 CAD 或求解器的数字，都必须先在该表有一行记录。

### 可直接作为 R0 初值的公开范围

| 特征 | R0 采用值或范围 | 证据含义 |
|---|---|---|
| 执行片有效长度 | 6–8 mm（初值 8 mm） | 专利实施例范围；不是 Mini 的保密精确尺寸 |
| 驱动频率 | 20–25 kHz（初值 22 kHz） | 8 mm 执行片实例的公开范围 |
| 膜片峰值位移 | 10–60 μm（初值 30 μm） | 专利范围；须靠系统级指标标定 |
| 顶腔高度 | 200–300 μm（初值 250 μm） | 公开实施例范围 |
| 底腔高度 | 位移 + 5–10 μm | 与膜片运动强绑定，不能随意取值 |
| 喷孔宽度/直径 | 200–300 μm（初值 250 μm） | 公开实施例范围 |
| 喷孔间距 | 400–600 μm（初值 500 μm） | 公开实施例范围 |
| 孔板开孔率 | 8–12%（初值 10%） | 公开实施例范围 |
| 孔板至热面冲击间隙 | 200–300 μm（初值 250 μm） | 公开实施例范围 |
| 喷孔出口速度 | 至少 30–35 m/s；探索 30–60 m/s | 公开性能阈值/实施例范围 |

### 必须标定、不能假装已知的量

- AirJet Mini 某一代真实膜片材料层厚、PZT 图形、电极和残余应力；
- 每一 cell 的精确数量、孔板图案、流道截面、封装泄漏；
- 实际驱动电压、波形、Q 值、阻尼和相位；
- 空气从热面离开后在整机里的真实排气路径；
- 真实芯片/TIM/均热板堆叠。

这些量先设为“待标定”，通过**同时匹配多个公开系统指标**缩小范围，而不是任选一个数字。

## 4. 复原顺序（不能跳步）

### R1：证据到 CAD（无求解）

输出：带尺寸、部件名称和证据编号的二维剖面；再生成参数化 3D 单 cell。

1. 从 `ledger.csv` 锁定 R0-A 初值。
2. 在 SpaceClaim/Creo/SolidWorks 建立“一个 cooling cell”：顶板、进气口、顶腔、中央锚点、膜片、底腔、孔板、喷孔、冲击间隙和四周排气空间。
3. 不能先画阵列。先把单 cell 的尺寸关系、运动空间和流道连通性画对。
4. 给每一个尺寸标注 `E`（公开直接证据）、`I`（由几何关系推断）、`C`（未来要标定）。

### R2：压电/结构模型（无 CFD）

输出：模态形状、谐响应曲线、位移场 `w(x,y,t)`。

1. 用 ANSYS Mechanical 或 COMSOL 建膜片 + 约束；先可用等效弹性板和规定中心锚定验证模态。
2. 再加入金属基底、压电层和电极的等效层；尚不能确定材料时，用“等效刚度 + 等效质量 + 等效阻尼”标定到 20–25 kHz、10–60 μm 的目标区间。
3. 输出的不只是频率，而是整个膜片位移场。CFD 需要它来驱动动态网格。

### R3：单 cell 全瞬态流动模型（无热）

输出：每周期 `m_dot(t)`、喷孔速度、腔压、回吸比例和净流量。

1. 由 R2 位移场驱动 Fluent 动态网格；不能直接用稳定速度入口替代，因为当前目标是复原膜片泵气机理。
2. 空气必须采用瞬态、可压缩理想气体；频率在超声段、腔体很薄，声学压力不可忽略。
3. 用 R0-A 的孔板位置与底腔高度，检查上冲程是否优先从顶腔补气、下冲程是否从喷孔喷出。
4. 以净流比、喷孔峰值速度、压力幅值和周期稳定性标定位移/阻尼/流道损失。

### R4：加入真实热路径的 CHT

输出：芯片—TIM—扩散板—冲击气流的周期温度场，以及系统热移除率。

热模型必须放在 R3 之后。否则你得到的可能只是“能散热的普通微射流”，不是被 AirJet 动力学约束的模型。

### R5：系统级校准与不确定性包络

输出：至少三个性能指标的匹配表与低/中/高参数包络。

优先校准次序：封装尺寸 → 频率/模态 → 喷孔速度/净流 → 背压 → 热移除/输入功。若一个参数组合只能匹配热量却严重偏离频率或背压，则拒绝该组合。
