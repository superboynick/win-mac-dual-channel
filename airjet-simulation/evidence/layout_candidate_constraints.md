# AirJet Mini 内部完整布局候选约束

状态：P0 工作顺序已冻结；尚未锁定真实 cell 数。

## 1. 不直接数宣传图

官方横截面的绿色波浪、蓝色箭头和顶部开口主要用于解释工作机理，可能没有按真实 cell 数绘制。它们只能证明“存在多个振动膜片和分布式进气/排气”，不能直接把图形数量写成产品内部数量。

## 2. 布局必须同时满足的硬约束

| 约束 | 表达 |
|---|---|
| 产品宽度 | `2 m_side + N_x p_x ≤ 27.5 mm` |
| 产品长度 | `m_inlet + m_exhaust + N_y p_y ≤ 41.5 mm` |
| 执行片尺寸 | 专利广义 4–10 mm；6–8 mm 为优选实施例，范围外侧候选需降低先验权重 |
| cell 壁/间隙 | 膜片到腔壁保留约 0.2–0.3 mm 量级 |
| 锚点 | 中央锚点宽度与膜片比例处于专利范围 |
| 开孔率 | 活动孔板区域 8–12% |
| 总功耗 | `N_cell P_cell + P_driver ≤ 1 W` |
| 压力与流量 | 分开验证低背压非零流量/热输运与最高压力能力；1750 Pa 对应流量条件未公开，不能假定该点仍有指定净流 |
| 热覆盖 | 喷孔活动区能解释完整热扩散面的温度分布 |
| 流道 | 所有 cell 都能从顶盖取气并向实际出口排气 |

## 3. 三个候选族

### Layout-L：少量大 cell

- 目标：测试公开专利中接近 8–10 mm 的大执行片是否能覆盖产品；
- 优先候选：宽度方向 2–3 列、长度方向 3–4 行；
- 需要证明：边缘活动区和总流量不会不足。

### Layout-M：中等 cell（主候选族）

- 目标：执行片约 6–8 mm；
- 优先候选：宽度方向约 3 列、长度方向 4–5 行；
- 需要证明：能为入口、排气歧管和 flex 留出空间，且总功耗不超限。

### Layout-S：较多小 cell

- 目标：测试更细分的驱动阵列能否解释均匀射流和官方图；
- 优先候选：宽度方向 3–4 列、长度方向 5 行以上；
- 需要证明：单 cell 功耗和制造/隔墙面积不会使有效面积过低。

这些行列数只是搜索范围，不是产品事实。

## 4. 候选筛选次序

1. **几何可装配**：外壳、进气、排气、flex、活动区互不冲突；
2. **结构可工作**：目标模态位于超声频段，位移不碰撞；
3. **功耗可闭合**：总功耗 ≤1 W；
4. **气动可闭合**：低背压净流方向/热输运、压力能力扫描、喷速下限先验和热移除共同自洽；1750 Pa 不作为已知流量工作点；
5. **热可闭合**：1 W 工况下净热 4.25 W、总热 5.25 W；
6. **图像一致**：候选布局投影与官方开口/截面比例不矛盾。

## 5. 需要从图片提取的尺寸比例

- 活动膜片/孔板区占产品长度和宽度的比例；
- 顶部每个进气开口的位置、长度、宽度与组数；
- 出口 spout 的宽度、厚度和相对热面高度；
- flex/驱动非活动区占用；
- 截面中顶腔、执行片、孔板、冲击通道和扩散板的厚度比例。

每项保存原始像素坐标、缩放基准和选择误差。P0 v1 统一采用包络角点 ±3 px、vent PCA 端点 ±2 px，并另计双视图 model-form 差异；其他未来特征必须单独定义误差，不能沿用一个模糊的 ±1–2 px。

## 6. 评分表

每个候选得到 0–5 分：

- `S_geometry`
- `S_image`
- `S_modal`
- `S_power`
- `S_flow`
- `S_thermal`
- `S_complexity`

硬约束失败直接淘汰；总分只用于在可行候选中排序。最终保留一个主布局和至少一个备选布局，直到获得实物或更强公开证据。

## 7. P0 评分定义与权重

评分表由 `build_layout_candidate_scores.py` 生成到 `layout_candidate_scores.csv`。每一个唯一几何用 `geometry_key` 去重；`family_tags` 只记录它来自 L/M/S 的哪个搜索入口，不能让同一几何重复参与排名。

| 分项 | 权重 | 5 分含义 | 0 分含义 | P0 可否打分 |
|---|---:|---|---|---|
| `S_geometry` | 15% | 完整 3D 包络、厚度、连通流道均通过并有稳健余量 | 越界、碰撞或断流 | 目前只有 2D A0 临时分 |
| `S_image` | 10% | 至少两个独立官方视图在测量误差内相容 | 与已确认的进排气拓扑冲突 | 暂留空；Gen1 只有透视渲染和示意剖面 |
| `S_modal` | 15% | 频率、振型、位移和应力对不确定性稳健 | 无可行模态或发生碰撞 | P2 前留空 |
| `S_power` | 15% | 含驱动器上界仍不超过 1 W | 功耗可信下界已超过 1 W | P2 前留空 |
| `S_flow` | 20% | 低背压非零流量、压力能力扫描、守恒和均匀度均通过 | 净流方向错误或压力能力失败 | P3/P4 前留空 |
| `S_thermal` | 20% | 多曲线点、热账户和能量误差均通过并留验证集 | 不能闭合 4.25 + 1 = 5.25 W | P5 前留空 |
| `S_complexity` | 5% | 满足功能后结构/求解最简 | 现有资源不可执行 | P0 只用 cell 数做代理 |

`weighted_score_100 = sum(weight_i * S_i / 5)`。未评分字段保持为空，不填 `0`，也不把已有权重重新归一化到 100%。另存 `score_coverage_pct`；当前可装入候选只有 geometry 与 complexity，覆盖率仅 20%。因此 CSV 的 17--19 分不是性能总分，更不是“真实布局概率”。

P0 的 2D `S_geometry` 只取宽/长最小剩余量：`>=4 mm -> 5`，`2--<4 -> 4`，`1--<2 -> 3`，`0--<1 -> 2`，`=0 -> 1`，负余量为 `0`。它仍未检查 2.8 mm 厚度预算、流体连通或部件碰撞。

## 8. 当前 A0 假设与淘汰含义

当前 `A0` 仅为可重复搜索假设：侧边余量 `1.0 mm`、进气加排气总预留 `5.0 mm`、cell 壁 `0.25 mm`、膜片平面投影暂按正方形代理。原始 34 个 family 组合去重后为 32 个唯一几何；其中 23 个在 A0 下可装入，9 个不装入。

`FAIL_CONFIG_A0` 只说明该组合不满足 A0，不是对真实产品的永久淘汰。只有当图像、CAD、结构、功耗、流动或热门槛给出独立失败证据时，才允许将候选升级为 `REJECTED`。

压力门槛必须分开：一组算例检查较低背压下确有非零净流和热输运；另一组扫描产品的 `1750 Pa` 压力能力。公开资料没有给出 1750 Pa 对应流量，所以不得规定该点必须达到一个虚构的净流量。

## 9. P0 工作候选，不是产品事实

- `M-3x4-7.0`：`PRIMARY-P0`，12 cells；A0 下宽/长余量 `4.00/7.75 mm`，方形膜片面积代理 `51.52%`。7 mm 位于专利 6--8 mm 优选族内部，适合作为第一套完整 CAD 试装。
- 唯一几何 `M+S-3x5-6.0`：`ALTERNATE-P0`，15 cells；余量 `7.00/5.50 mm`，面积代理 `47.32%`。它用于检验更多 cell 对流量均匀度与功耗的影响。
- `L-2x4-8.0` 和 `S-3x5-5.5`：保留为低 cell 与小 cell 的 model-form sentinel，不进入当前主/备选排名。

以上顺序只决定下一轮 CAD/仿真实验顺序，不表示对真实 Mini 内部 cell 数的概率判断。顶部四组进气槽不等于四个 cell，官方剖面中的绿色波形数量也不等于真实膜片数量。

## 10. 硬门槛顺序

1. `G0-EVIDENCE`：输入均有 D/P/I/C/U、来源、范围与可调性；
2. `G1-ENVELOPE`：在明确 assumption set 下装入 27.5 x 41.5 mm；
3. `G2-TOPOLOGY`：顶部进气、上下腔路径、冲击通道和单侧 integrated spout 连通；
4. `G3-THICKNESS`：顶盖、腔体、膜片、孔板、冲击间隙与扩散面闭合到 2.8 mm；
5. `G4-CAD`：无碰撞、盲孔、孤立空气域，孔数/孔形/开孔率自洽；
6. `G5-POWER`：`N_cell P_cell + P_driver <= 1 W`；
7. `G6-FLOW`：低背压净流、压力能力扫描、cell starvation 与守恒分别通过；
8. `G7-THERMAL`：净热、自热、总热和能量守恒共同闭合。
