# 第1课：CFD 基础与 AirJet 仿真

## 什么是 CFD？

CFD（Computational Fluid Dynamics，计算流体力学）是用计算机模拟流体（气体、液体）流动的技术。它把连续空间切成大量小格子（网格），在每个格子里求解流体力学方程。

### 三大守恒方程

每个网格单元都要遵守三个守恒：

1. **质量守恒**（连续性方程）：流入 = 流出，流体不会凭空消失
2. **动量守恒**（Navier-Stokes 方程）：力 = 质量 × 加速度，考虑压力、粘性、重力
3. **能量守恒**：热量进出平衡（CHT 时需要）

### AirJet 为什么需要 CFD？

AirJet 是固态散热芯片——没有风扇叶片。它用 **压电膜片振动** 驱动空气：
- 膜片上下振动 → 腔内压力变化 → 空气从 972 个微孔吸入/喷出
- 喷出的高速气流撞击散热器表面 → 带走热量

CFD 能回答的问题：
- 972 个微孔的流速分布是否均匀？
- 哪个孔的流量最大/最小？为什么？
- 出口气流的速度和方向？
- 整体散热能力（热阻）？

## 从 CAD 到 CFD

### Step 1: 几何建模（SpaceClaim）

把 AirJet 的物理结构画成 3D 模型。我们的模型包含：
- 12 个单元（cell），每个 81 个微孔（throat），共 972 个
- 底部腔体 → 顶部腔体 → 微孔板 → 散热器表面
- 关键尺寸：孔径 0.25mm，孔长 0.10mm（C016 参数，待验证）

### Step 2: 流体域提取（Boolean）

CAD 画的是固体，CFD 需要的是 **流体能经过的空间**。
用 Boolean 运算从固体中"挖出"流体域。

我们的挑战：0.02mm→0.15mm 重叠量调整让 Boolean 成功合并。

### Step 3: 网格划分（Meshing）

把流体域切成小格子。Fluent watertight 工作流：
1. Import STEP 几何
2. 识别边界（inlet, outlet, wall, throat）
3. Surface mesh（面网格）
4. Describe geometry（告诉 Fluent 这是流体域）
5. Volume mesh（体网格）

### Step 4: 求解（Solver）—— 未来阶段

设定边界条件（进口压力、出口压力等），让 Fluent 迭代求解。

## 关键概念

### 网格独立性（Mesh Independence）

网格越细 → 结果越精确 → 计算量越大。
需要在"够细"和"能跑"之间找平衡。

做法：从粗到细做 3 套网格（3.5万 → 50万 → 200万），看结果收敛。

### 正交质量（Orthogonal Quality）

衡量网格单元"方不方"的指标：
- 1.0 = 完美正方体
- 0.5 = 还行
- <0.1 = 太差了，求解会发散

我们目前 C5 粗网格：min OQ = 0.49~0.57

### 边界条件类型

| 边界 | Fluent Type | 含义 |
|------|-------------|------|
| 4 个入口 | velocity-inlet | 已知流速 |
| 1 个出口 | pressure-outlet | 出口通大气 |
| 972 个微孔 | interior | 空气通过 |
| 固体壁面 | wall | 空气不能穿过 |
