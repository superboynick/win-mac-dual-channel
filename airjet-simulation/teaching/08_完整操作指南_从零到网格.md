# 第8课：完整操作指南——从零到网格

> 适合：已有 ANSYS Student 2026 R1 + Python 3.12 的 Windows 用户

## 前置条件

1. ANSYS Student 2026 R1 安装在 `D:\ansys\ANSYS Inc\ANSYS Student\v261`
2. Python 3.12 虚拟环境：`C:\Users\admin\AppData\Local\AirJetAnsysAutomation\.venv`
3. 安装 PyFluent：`pip install ansys-fluent-core`
4. 克隆仓库：`git clone git@github.com:superboynick/win-mac-dual-channel.git`

## 完整操作步骤

### Step 1: 验证安装

```powershell
# 检查 ANSYS
Test-Path "D:\ansys\ANSYS Inc\ANSYS Student\v261\scdm\SpaceClaim.exe"

# 检查 Python
C:\Users\admin\AppData\Local\AirJetAnsysAutomation\.venv\Scripts\python.exe --version

# 检查 Git
git status
```

### Step 2: 运行离线测试（无 ANSYS）

```powershell
cd C:\Users\admin\win-mac-dual-channel
python airjet-simulation/automation/ansys/contracts/test_v03_finite_throat_contract_v1.py
python airjet-simulation/automation/ansys/test_run_v03_continuous_mesh_006.py
```

两个都输出 `PASS_ALL` 才能继续。

### Step 3: 运行 Stage 1（CAD 几何）

```powershell
python airjet-simulation/automation/ansys/run_v03_continuous_fluid_006.py
```

这会在 SpaceClaim 中：
- 读参数
- 构建 12 单元 × 972 孔几何
- Boolean 合并
- 导出 STEP

产出物在 `airjet-simulation/logs/p1_cad_run/`。

### Step 4: 运行 Stage 2（网格）

```powershell
python airjet-simulation/automation/ansys/run_v03_continuous_mesh_006.py
```

这会在 Fluent 中：
- Import STEP
- Surface mesh
- Volume mesh
- 导出 .msh.h5

产出物：`v03_continuous_volume_mesh.msh.h5`（约 8 MB）。

### Step 5: 验证网格

检查网格质量：
- cells: 35,000-40,000
- OQ min: > 0.15
- free_face: 0
- multi_face: 0

完整报告：`airjet-simulation/logs/p1_cad_run/V03_CONTINUOUS_MESH_RUN_SUMMARY.json`

## 常见问题

### Q: "MCP server not found"
A: 检查 `codex-skills/airjet-ansys-automation/scripts/airjet_ansys_mcp.py` 存在

### Q: "ANSYS license not available"
A: Student 版自动使用本地许可。确保没有其他 ANSYS 进程运行

### Q: "Volume mesh only 13.5 mm³"
A: Geometry Boolean 连接问题。检查 C5 overlap 参数

### Q: "Cell limit exceeded"
A: Student 版限制 1M cells。需要 Academic Research License
