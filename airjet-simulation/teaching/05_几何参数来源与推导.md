# 第5课：AirJet 几何参数来源与推导

## 参数来源层级

我们按证据强度给参数分级：

| 级别 | 含义 | 例子 |
|------|------|------|
| A | 专利/官方数据表直接给出 | 12 cells, 3×4 排列 |
| B | 从 A 级数据推导 | 972 个孔（12×81） |
| C | 半推导/工程估算 | 膜片厚度、排气槽尺寸 |
| D | 完全假设/候选值 | PZT 材料属性 |

## 核心几何参数

### 确定的（A/B 级）

| 参数 | 值 | 来源 |
|------|-----|------|
| 单元数 | 12 | Frore 产品资料 |
| 单元排列 | 3×4 | 拆解照片 |
| 每单元孔数 | 81 | 显微镜图像计数 |
| 总孔数 | 972 | 12 × 81 |
| 孔径 | 0.25 mm | 专利 US2021/XXXX（喉道直径） |
| 产品厚度 | ~2.8 mm | 产品规格书 |
| 产品尺寸 | 27.5 × 41.5 × 2.8 mm | 产品规格书 |

### 推导的（C 级）

| 参数 | 值 | 推导方法 |
|------|-----|----------|
| 喉道长度 C016 | 0.10 mm（候选） | 从蚀刻工艺估算，初始值 |
| 膜片尺寸 | 7 × 7 mm | 从单元间距反推 |
| 膜片厚度 | 0.10 mm | 从压电片标准厚度估算 |
| 腔体深度 | 0.25-0.50 mm | 从总厚度减去各层 |
| 排气方向 | Y+ | 产品资料显示侧向排气 |

### 不确定的（D 级）

| 参数 | 候选值 | 不确定性 |
|------|--------|----------|
| PZT 材料 | PZT-5H | 常见压电材料，需确认 |
| 膜片位移 | 10-50 μm | 取决于驱动电压 |
| 工作频率 | 20-24 kHz | 超声范围，需确认 |

## Z 向分层（从底部到顶部）

```
Z = 1.2675 mm  — 底部（Bottom Cover 底面）
Z = 1.5175 mm  — 底部腔体顶 = 喉道入口
Z = 1.6175 mm  — 喉道出口 = 顶部腔体底
Z = 2.8000 mm  — 顶部（Top Cover 顶面）
```

各层厚度：
- Bottom Chamber: 0.25 mm
- Orifice Plate: 0.10 mm（C016 候选值）
- Top Chamber: 0.25 mm
- Top Cover: 0.50 mm

## 喉道布局（972 个孔如何排列）

- 每单元 81 个孔 = 9×9 阵列
- 孔间距：0.70 mm（最小间距，从参数推导）
- 每孔坐标由 `variant_02_m_3x4_7_0_r50_balanced.json` 精确给出
- SHA256 哈希保证了坐标的不可篡改性

## 如何验证参数

1. **专利交叉验证**：多个专利对同一参数的值是否一致
2. **物理约束**：厚度加起来等于总厚度
3. **拆解图像**：从拆解照片测量关键尺寸
4. **仿真反推**：调整候选参数直到仿真结果匹配公开性能数据

## 参数不确定性对仿真的影响

| 参数 | 如果不准会导致... |
|------|-------------------|
| 喉道长度 | 流量计算偏差 |
| 膜片位移 | 入口速度偏差 |
| 孔径 | 喉道马赫数偏差 |
| 腔体深度 | 压力分布变化 |

这就是为什么 C016（喉道长度）被标记为 `uncertainty_scan: REQUIRED_LATER_NOT_RUN`。
