# 操作手册 00：从公开证据建立 AirJet R0-A 单 cell CAD

> **P2/P3 子模型参考，当前不得先执行。** 必须先完成整机主规划的 P0/P1；本手册中的尺寸均为专利候选，不是 Mini 产品锁定值。当前 CAD 主手册是 `manuals/01_FULL_PRODUCT_CAD.md`。

适用阶段：仅在 P1 整机布局确定后，用作 P2/P3 cell 截取参考。
目标：得到一个**流道完整、尺寸可追溯、可用于后续结构和流体求解**的单 cell 三维 CAD。

## 0. 开始前

打开：

1. `AIRJET_RECONSTRUCTION_PLAN.md`
2. `evidence/airjet_reconstruction_ledger.csv`
3. 本地资料 `AirJet_research/patents/US12137540B2.pdf`

软件：Windows 上优先 SpaceClaim；没有时可用 SolidWorks/Creo/Fusion。单位固定为 **mm**。

## 1. 建一个参数表，不直接画死尺寸

在 CAD 的参数管理器录入：

| 名称 | 值（mm） | 证据 ID |
|---|---:|---|
| `L_mem` | 8.000 | E02 |
| `t_mem_eff` | 0.275 | E04 |
| `h_top` | 0.250 | E05 |
| `A_tip` | 0.030 | E06 |
| `h_bottom` | 0.040 | E07 = `A_tip + 0.010` |
| `w_vent` | 0.750 | E08 |
| `d_orifice` | 0.250 | E09 |
| `s_orifice` | 0.500 | E10；专利称 separation `s`，图义确认前不写成中心节距 |
| `g_jet` | 0.250 | E12 |
| `w_anchor` | 2.250 | E14 |
| `d_wall` | 0.250 | E15 |

`t_mem_eff` 是等效复合膜厚，不等于已知真实 PZT 层堆叠。不要在模型树中写“AirJet actual”。

## 2. 先画 2D 纵剖面

在 X–Z 平面从上至下依次画：

1. 顶板和宽度 `w_vent` 的进气口；
2. 高度 `h_top` 的顶腔；
3. 中央锚点和以其为中心支承的膜片；膜片两侧必须留为可振动悬臂；
4. 高度 `h_bottom` 的底腔；
5. 孔板；
6. 孔板下方高度 `g_jet` 的冲击间隙；
7. 冲击面和至少一侧可连通的排气空间。

在草图旁标注 E01–E15。对无直接证据的外壁厚、孔数、排气道长度标注 `C-01`、`C-02` 等待标定编号。

## 3. 建为 3D 单 cell

1. 以 `L_mem=8 mm` 建立膜片平面；深度先取 `8 mm`，形成近似正方 cell。
2. 中央锚点沿中心轴布置，宽度 `w_anchor`；两侧膜片仍能上下变形。
3. 顶腔与底腔必须在膜片周边相通；不要错误地把膜片四周全部封死。
4. 在孔板建立阵列孔。先让孔板开孔率接近 10%；孔数不是证据锁定值，记录为 `C-02`。
5. 在孔板下方建 jet channel；四周加足够大的侧向排气空间，避免出口紧贴喷孔。

## 4. 必做的四项几何检查

- `Check A`：从顶板进气口能连续到顶腔、膜片外围、底腔、每个孔、冲击间隙和排气出口。
- `Check B`：任何膜片位置都不接触顶板或孔板；初值下底腔最小余量为 10 μm。
- `Check C`：孔宽、专利间隔 `s`、实际 CAD 节距、孔数和开孔率计算一致；先确认 `s` 的图义，不要默认它是中心距。
- `Check D`：所有“真实 AirJet 未公开”的尺寸都有 `C-xx` 标签，没有伪装成证实值。

## 5. 导出与记录

保存为 `geometry/R0A_single_cell_v001.<native CAD extension>`，并导出：

- 一张带尺寸和 E/C 标签的纵剖面 PNG/PDF；
- STEP 文件；
- `MODEL_ANNOTATIONS.md` 中的 R0A-001 条目。

不要在这一步生成网格、设温度、设入口速度或建立动态网格。先让我检查剖面及流道连通性，再进入 R2。
