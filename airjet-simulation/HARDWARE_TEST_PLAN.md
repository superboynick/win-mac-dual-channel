## 2. 数据处理

### 2.1 尺寸校准
```
实测参数 → 对比当前CAD参数 → 计算偏差 → 更新参数注册表
```
- 将每个实测值录入 `parameters/full_product_parameter_registry.csv`
- 标记证据等级从 B/C 升级为 A
- 计算偏差百分比

### 2.2 热数据
```
热像图 → 提取关键点温度 → 对比CFD仿真温度 → 误差分析
```
- ZBOX满载时CPU表面温度
- ZBOX外壳温度分布
- AirJet出风口温度

### 2.3 声学数据
```
录音 → FFT分析 → 提取主频 → 对比膜片驱动频率
```
- AirJet工作频率预期：~20-40 kHz（超声，人耳不可闻）
- 如果可闻：可能是谐波或机械噪声

## 3. 仿真调整计划

### 3.1 Phase 1：参数更新（收到硬件后24小时内）
1. 用实测尺寸更新所有C级参数→A级
2. 重新运行SpaceClaim producer（v03_continuous_fluid_producer.py）生成校准几何
3. 运行save_mesh6.py生成校准网格
4. 对比新旧网格指标（cell count, OQ, volume）

### 3.2 Phase 2：边界条件校准
1. 根据实测进出口面积计算实际流速
2. 根据功耗推算热流密度
3. 更新CFD boundary conditions
4. 运行solve_cfd6.py→得到校准后的流场/温度场

### 3.3 Phase 3：仿真-实验对比
1. 提取CFD仿真中的关键点温度
2. 与热像图实测温度对比
3. 计算误差：|T_sim - T_exp| / T_exp
4. 分析误差来源（网格分辨率、BC不确定性、材料属性）

### 3.4 Phase 4：迭代优化
1. 如果误差>20%：调整BC、材料属性、网格
2. 如果误差<10%：锁定模型参数
3. 运行网格独立性验证（coarse/medium/fine）
4. 完整求解（CHT、FSI）
