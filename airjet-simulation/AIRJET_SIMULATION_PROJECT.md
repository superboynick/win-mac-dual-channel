## 1. 拟定论文题目与核心问题

### 推荐主题

**压电 MEMS 驱动微射流冲击冷却中喷嘴—基板间距与热扩散能力的协同优化：面向 AirJet 类薄型主动散热器的数值研究**

这是“AirJet 类（AirJet-inspired）”研究，而不是宣称完整复现 Frore AirJet。这样既能使用公开专利和官方性能资料设定边界，也能把创新点放在可计算、可验证的传热流动规律上。

### 主研究问题

在总输入功、可用空间和噪声/压损受限时，喷嘴出口的脉冲微射流如何通过下列因素共同决定散热效果？

1. 喷嘴—受热面距离 `H/D`；
2. 喷嘴直径、孔距和阵列布局；
3. 热扩散板材料与厚度；
4. 热界面层（TIM）热阻；
5. 排气通道对回流、横向串扰和压损的影响；
6. 稳态等效射流和真实周期脉冲射流之间的差异。

### 可检验假设

- 对固定气动输入，`H/D` 存在使 `Tmax` 最低而非单调降低的区间：间隙太小会限制排气，太大则导致射流衰减。
- 高导热扩散板可降低热点温度，但在热源面积较大或 TIM 热阻很高时，收益会饱和。
- 周期脉冲会改变瞬态壁面剪切与涡结构；其平均换热不必等于同平均质量流量的稳态射流。
- 均匀阵列未必是热点散热的最优布局；热点附近的局部孔径/孔距调整可能以较小的压损代价改善 `Tmax`。

## 3. 总体技术路线

```text
公开结构与规格资料
        │
        ▼
0D/1D 气动估算（流量、Re、压力、功率上限）
        │
        ▼
阶段 A：3D 稳态等效微射流 CHT 基线 ── 网格/守恒/热平衡验证
        │
        ├── 参数扫描：H/D、D、pitch、扩散板、TIM、排气道
        ▼
阶段 B：单 cell 周期瞬态 CFD（规定膜片位移或速度）
        │
        ├── 与等效稳态结果比较：平均 h、Tmax、Δp、流量
        ▼
阶段 C：阵列/热点优化（周期扇区或局部阵列）
        │
        ▼
可选阶段 D：压电模态/谐响应 → 单向流固耦合
        │
        ▼
仅在必要时：双向 FSI
```

**首篇论文建议止步于阶段 C。** 阶段 D 是增强可信度的后续工作；全产品、全阵列双向 FSI 的未知参数与计算成本过高，不能作为起始工作。

## 4. 几何与物理模型定义

### 4.1 阶段 A：等效射流 CHT（第一个必须跑通的模型）

求解域由五部分组成：

| 部分 | 建模方式 | 目的 |
|---|---|---|
| 喷嘴/孔板 | 圆孔或等面积出口，`D`、`pitch` 参数化 | 形成可控射流 |
| 冲击间隙 | `H` 参数化，建议覆盖 `H/D=1–10` | 研究冲击、横向排气与回流 |
| 流体域 | 空气，低马赫数可压缩或不可压缩 | 求速度、压力和温度场 |
| 固体域 | 热源/芯片、TIM、扩散板 | 共轭传热与热点温度 |
| 排气域 | 四周开口或显式排气道 | 避免把封闭域误当真实 AirJet |

最小可用几何是“单喷嘴 + 正方形周期单元 + 热源斑块 + 扩散板”。先证明它正确，再增加到 `3×3` 喷嘴和真实边界排气。

### 4.2 阶段 B：周期脉冲单 cell

在阶段 A 几何上增加驱动腔和移动壁面。膜片不必先建压电多层实体，可采用：

`x(t) = A sin(2π f t)`，或 `v(t) = 2π f A cos(2π f t)`。

其中 `A`（振幅）和 `f`（频率）是待标定参数。用动态网格/ALE 使腔体体积周期变化，观察一个或多个完整周期的流量、压力与壁温响应。

### 4.3 阶段 D：单向耦合（可选）

用 ANSYS Mechanical/COMSOL 求压电膜片的模态和谐响应，输出位移时程与有效振型；将该位移映射到 CFD 动态网格。此处先不让流体压力反作用回结构，即为单向耦合。

### 4.4 不要一开始建的内容

- 完整商业 AirJet 的精确堆叠层、材料厚度、驱动波形；公开资料不足。
- 全阵列、全产品、双向 FSI。
- 复杂湍流、声学、压电、电路和颗粒污染的全耦合。

## 5. 控制方程、算法与软件实现

### 5.1 方程组

流体采用质量、动量和能量守恒：

`∂ρ/∂t + ∇·(ρu) = 0`

`∂(ρu)/∂t + ∇·(ρuu) = -∇p + ∇·τ`

`∂(ρE)/∂t + ∇·[u(ρE+p)] = ∇·(keff∇T)`

固体区域求解导热方程：

`ρs cps ∂T/∂t = ∇·(ks ∇T) + q'''`

流固界面满足温度连续和热流连续。若阶段 B 中入口马赫数、压力波动或密度变化显著，应使用理想气体可压缩形式；否则阶段 A 可先使用常密度低马赫数模型，并用可压缩算例做敏感性检查。

### 5.2 数值方法（推荐 Fluent 路线）

| 项目 | 阶段 A 推荐 | 阶段 B 推荐 | 原因 |
|---|---|---|---|
| 求解器 | Pressure-Based, Steady | Pressure-Based, Transient | 适合低速内部/冲击流；后者支持周期解 |
| 压速耦合 | SIMPLE 或 Coupled | PISO 或 Coupled | 稳态稳健；瞬态对每步压力校正更好 |
| 能量方程 | 开启 | 开启 | 必需 |
| 湍流 | Laminar 起步；SST k-ω 对照 | SST k-ω URANS | 微尺度 Re 可能低；近壁/射流分离要对照 |
| 空间离散 | 二阶压力、二阶动量/能量 | 二阶隐式时间 + 二阶空间 | 降低数值扩散 |
| 动态网格 | 不用 | Layering + smoothing/remeshing，或 overset | 实现移动膜片 |
| 材料 | air + Si/Cu/AlN + TIM 等效层 | 同左 | 与 CHT 相一致 |

不应先假定“必须湍流”。先根据 `Re = ρUD/μ` 给出 Re，再同时运行 laminar 与 SST k-ω；若主要结论对模型选择敏感，论文必须报告这一不确定性。

### 5.3 可替代软件

- **ANSYS Fluent + Mechanical**：推荐 Windows 主路线，动态网格、CHT、参数化和后处理最直接。
- **COMSOL Multiphysics**：适合膜片压电、声学与流固耦合原型，但大规模瞬态 CFD 的资源压力更高。
- **OpenFOAM**：可复现与批量扫描强；需要自行处理动态网格、CHT 和稳健性，适合作为第二路线而非第一次建模。

### 5.4 收敛与周期稳定判断

- 阶段 A：连续残差只是辅助；要求质量不平衡 < 0.5%，固体/流体热量差 < 1%，`Tmax` 和 `Δp` 在 500 次迭代内变化 < 0.1%。
- 阶段 B：至少计算 10–20 个周期；当相邻两个周期的周期平均质量流量、`Tmax`、`Δp` 差异均 < 1% 时，视为周期稳定。
- 时间步：先取每周期 100–200 步，即 `Δt = 1/(100f)` 到 `1/(200f)`；再做时间步独立性检验。

### 6.2 首轮几何与热边界扫描

| 变量 | 符号 | 基线 | 首轮范围 | 目的 |
|---|---|---:|---:|---|
| 喷嘴直径 | `D` | 0.30 mm | 0.15–0.60 mm | 射流尺度与压损 |
| 孔距 | `P` | 1.00 mm | `2D–5D` | 射流干扰/覆盖率 |

| 扩散板厚度 | `t_sp` | 0.50 mm | 0.20–1.50 mm | 横向导热能力 |
| TIM 厚度 | `t_TIM` | 50 μm | 20–150 μm | 接触热阻 |

| 热源功率 | `Q` | 5 W | 2–15 W | 热负荷适用区间 |
| 出口/排气压力 | `p_out` | 0 Pa gauge | 0–100 Pa gauge | 背压敏感性 |

### 6.3 气动输入：两套可比边界

为保证结果不依赖某一未知产品驱动，阶段 A 同时报告：

1. **定速度/定质量流量**：喷嘴入口 `Uin = 5, 10, 20, 40 m/s`（或等效 `ṁ`）。优点是方便比较几何。
2. **定压差**：腔体到出口 `Δp = 50, 100, 200, 500 Pa`。优点是更接近气动驱动约束。

每个工况都计算：`Re`、`ṁ`、`Δp`、气动功率近似 `Pfluid = Δp × Vdot`，以及总热阻 `Rth=(Tmax-T∞)/Q`。

### 6.4 阶段 B 驱动参数（待标定）

| 参数 | 符号 | 探索范围 | 状态 |
|---|---|---:|---|
| 膜片频率 | `f` | 5–40 kHz | 待公开资料/模态结果校准 |
| 峰值振幅 | `A` | 1–20 μm | 待校准 |
| 腔体高度 | `hc` | 0.1–1.0 mm | 参数化近似 |
| 波形 | — | 正弦；后续可加入非正弦 | 初始假设 |

频率与振幅不能凭印象写成“AirJet 实际值”。论文中应写成参数化激励，并把灵敏度结果作为贡献。

## 7. 网格、边界层和独立性验证

### 7.1 网格策略

- 喷嘴出口、冲击点、近壁层、排气狭缝局部加密。
- 喷嘴直径方向至少 20–30 个单元；初始可用 24，最终用三套网格验证。
- 若使用 SST k-ω：优先使受热面和喷嘴内壁 `y+≈1`；不要混用“壁函数”预期和低 `y+` 网格而不说明。
- 若 laminar：仍需解析速度/温度边界层，使用足够近壁分层。
- 网格质量指标：最大 skewness < 0.85，orthogonal quality > 0.15；动态网格应在整个周期检查负体积。

### 7.2 网格独立性

至少三档（coarse/medium/fine），比较 `Tmax`、面积平均 `h`、`Δp` 与 `ṁ`。当 medium→fine 的关键输出差异小于 2–3%，使用 medium 进行参数扫描、fine 做代表性确认。可用 GCI（Grid Convergence Index）报告网格不确定度。

### 7.3 时间步独立性

对阶段 B 至少比较 `100`、`200`、`400` steps/period。用最后两个周期的周期平均 `Tmax`、`h`、`ṁ` 和相位峰值对比；差异 < 2% 才固定时间步。

## 8. 验证、确认与不确定性

### 验证层次

| 层次 | 要回答的问题 | 证据/方法 |
|---|---|---|
| 代码/设置确认 | 方程、单位、边界、材料是否设置正确？ | 守恒检查、简化解析解、日志 |
| 数值验证 | 网格和时间步会不会改变结论？ | GCI、三网格、三时间步 |
| 物理验证 | 模型是否重现一般射流/合成射流规律？ | 与基础论文的 `Nu/Re`、速度/温度趋势比较 |
| 系统校核 | 输出是否与公开 AirJet 系统量级相容？ | 官方资料/教程，只作量级校核 |

没有公开的 AirJet 内部几何或流量时，不能声称“验证了真实 AirJet 内部流场”。更准确的说法是“针对公开结构启发的参数化微射流模型完成了数值验证，并与公开系统级量级交叉校核”。

### 不确定性清单

- 未公开的腔体尺寸、孔径、驱动频率、振幅和排气细节；
- MEMS 振幅受制造、温升、阻尼和驱动电路影响；
- TIM 接触热阻和芯片热流密度；
- 湍流转捩与低 Re 微射流的模型形式；
- 环境背压与系统封装泄漏。

对每一项给出低/中/高三档，而不是只报告单一“最佳”参数。

## 9. 评价指标与优化方法

### 必报输出

`Tmax`、平均温度 `Tavg`、温度均匀性 `σT`、总热阻 `Rth`、平均/局部换热系数 `h`、Nusselt 数 `Nu`、压损 `Δp`、质量流量 `ṁ`、气动功率 `Pfluid`、泵功/总输入功的估算、噪声代理量（压力脉动 RMS）。

### 多目标目标函数

不要只最小化 `Tmax`。可采用：

`min [Tmax, Rth, Δp, Pfuid]`，约束 `Q`、封装高度、最大允许温度与可用面积。

首轮用全因子或 Latin Hypercube Sampling（LHS）扫描；第二轮用响应面/高斯过程代理模型，然后做 NSGA-II 多目标 Pareto 优化。输出不应只是一组最优数，而要展示 Pareto 前沿：降低温度需要付出多少压损/功率。

### 推荐 DOE 顺序

1. 单因素 sanity check：`H/D`、`D`、`Q`。
2. `H/D × D × P/D` 的 3 水平全因子/部分因子。
3. 加入扩散板材料、厚度和 TIM 热阻。
4. 对热点工况比较均匀阵列与局部加密阵列。
5. 仅对 Pareto 候选进行阶段 B 的脉冲瞬态复算。

### Fluent 阶段 A 的操作次序

1. 建立单喷嘴或 `3×3` 阵列的流体域、固体扩散板、TIM 和热源。
2. Named Selections：`jet_inlet`、`exhaust`、`heated_chip`、`fluid_solid_interface`、`symmetry/periodic`。
3. 导入网格，先检查单位、体积和质量质量。
4. 开启 Energy；设置 air、固体材料和 CHT 接口。
5. 使用 `velocity-inlet` 或 `pressure-inlet`，四周 `pressure-outlet`；芯片用定功率/均匀热流。
6. 先跑 laminar，再复制为 SST k-ω 对照；保存 case 与 journal。
7. 导出每个工况的 `Tmax`、`Tavg`、`Δp`、`ṁ`、热通量平衡、残差和网格统计到 CSV。
8. 通过 `checklists/stage-gates.md` 的阶段 A 门槛后，才开始扫描。

### 阶段 B 增量实现

1. 在阶段 A 上游加入腔体和膜片移动壁面。
2. 用正弦 profile/UDF 或 table 输入 `v(t)`；初期不要同时接 Mechanical。
3. 开启 transient、二阶隐式时间、PISO；每周期至少 100 步。
4. 先使用两个周期调试网格，再跑到周期稳定。
5. 提取一个周期的 `ṁ(t)`、`Δp(t)`、`Tmax(t)` 和壁面 `h(t)`；与等效稳态结果比较。

## 12. 首两周可执行任务

### 第 1–2 天：定义与资料锁定

- 跑通一个 `Uin=10 m/s`、`Q=5 W` 工况。

### 第 6–8 天：数值验证

## 13. 版本管理与跨机约定

- 每个可复现的模型版本都有一个简短 ID，例如 `A01_singlejet_laminar`。
- Git 提交只包含：journal/UDF、参数 CSV、Python 脚本、图表源、模型设置说明、结果摘要。
- 不提交 `.cas/.dat`、大网格、瞬态场、许可证文件和商业 PDF；它们应保存在本地数据盘，且在 `RUN_LOG.md` 记录绝对路径或相对数据盘路径。
- 每次运行都写：软件版本、求解器、网格数、时间步、残差、守恒误差、CPU 核数、内存、运行时间。
- 在任一机器开始前 `git pull`；完成一段工作后 `git add`、`git commit`、`git push`。遇到分支分叉或脏工作区停止，不自动覆盖。

## 14. 本阶段的“完成”定义

第一阶段完成不等于跑出一张漂亮云图。必须同时满足：

- 有一份可复现的参数化几何/网格说明；
- 一个 CHT 基线算例能稳定收敛；
- 质量与能量守恒满足第 5.4 节门槛；
- 做完网格独立性，且记录不确定度；
- 至少得到 `H/D` 与一个几何变量的趋势；
- 对每一个结论同时报告温度收益和压损/功率代价；
- 明确哪些参数是公开证据、哪些是假设、哪些已校准。

做到这里，就拥有可以支撑论文方法与结果章节的可靠起点；之后再用阶段 B 回答“MEMS 脉冲驱动究竟会怎样修正稳态设计结论”。
