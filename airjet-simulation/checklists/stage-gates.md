# AirJet 仿真阶段门槛

> **状态：旧优化路线清单，当前停用。** 整机复原使用 `full_product_stage_gates.md`。

## Gate 0：立项

- [ ] 本地 AirJet 资料目录可访问，且主项目文件已读完。
- [ ] 论文题目写的是 AirJet 类/公开结构启发模型，不声称完整复刻商业产品。
- [ ] 所有初始参数已填入 `parameters/baseline_parameters.csv`，未知参数标为 assumed 或 unverified。
- [ ] 已选择首算几何：推荐单孔周期单元。

## Gate A：稳态 CHT 基线可用

- [ ] 几何单位、热源功率、喷嘴面积和出口面积已手工复核。
- [ ] 流体、固体、TIM、流固界面和排气边界均已显式设置。
- [ ] 质量不平衡小于 0.5%。
- [ ] 输入功率与流体/固体导出热量差小于 1%。
- [ ] `Tmax`、`Delta p` 在后 500 迭代内变化小于 0.1%。
- [ ] 保存 journal/设置说明，能够从零复跑。

## Gate B：数值可信

- [ ] 完成粗、中、细三档网格并记录单元数和质量指标。
- [ ] medium 到 fine 的 `Tmax`、`Delta p`、平均 `h` 差异不超过 2–3%，或已报告 GCI。
- [ ] 至少比较 laminar 与 SST k-omega；选择理由写入运行日志。
- [ ] 速度场、温度场和排气流线没有明显非物理解。

## Gate C：参数规律

- [ ] 至少完成 `H/D` 四点和一个几何变量的扫描。
- [ ] 每个工况已导出 `Tmax`、`Rth`、`Delta p`、`mdot`、`Nu/h` 和计算资源信息。
- [ ] 结论同时报告温度收益与压损/功率代价。
- [ ] 图表和 CSV 可以由脚本重新生成。

## Gate D：瞬态膜片模型（可选）

- [ ] 在 Gate A–C 全部完成之后再进入。
- [ ] 移动壁面位移/速度函数及其频率、振幅来源已记录。
- [ ] 动态网格全过程无负体积、无严重质量损失。
- [ ] 做过 100/200/400 steps-per-period 时间步独立性比较。
- [ ] 相邻两个周期的周期平均输出差异小于 1%。
- [ ] 不把未标定的频率/振幅解释为真实产品内部参数。
