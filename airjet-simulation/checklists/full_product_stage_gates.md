# AirJet Mini 整机复原阶段门槛

## P0 产品证据冻结

- [ ] Mini 数据表所有尺寸、功耗、散热和曲线已数字化。
- [ ] 官方图的比例测量保留原图、标尺和误差。
- [ ] 核心专利图与产品图建立部件对应表。
- [ ] 所有输入参数进入 full product registry，且 `evidence_class` 明确为 D/P/I/C/U，并记录范围/不确定度、推导父项和是否可调。
- [ ] Mini、G2、PAK 的用途边界已写清。

## P1 完整产品 CAD

- [ ] CAD 外包络严格为 27.5 × 41.5 × 2.8 mm。
- [ ] 顶盖、侧框、活动阵列、孔板、冲击通道、排气歧管和热扩散面全部存在。
- [ ] 不是单 cell 或无限周期模型。
- [ ] Layout-L/M/S 均能装入同一外壳。
- [ ] 全部进气路径和排气路径连通。
- [ ] 孔宽、孔间隔/节距定义、孔数、活动面积和开孔率自洽；专利 `separation s` 的几何含义有图号依据。
- [ ] 完整流体体积提取成功。

## P2 单元结构校准

- [ ] cell 几何来自 P1 主布局，不另造尺寸。
- [ ] 频率、振型、振幅和功耗同时报告。
- [ ] 材料层和阻尼的不确定性有范围。
- [ ] 单 cell 功耗乘以 cell 数不违反 1 W 整机预算。
- [ ] 位移场可导出给 CFD。

## P3 单元瞬态 CFD

- [ ] 可压缩瞬态和动态网格已验证。
- [ ] 至少三档网格和三档时间步。
- [ ] 相邻周期平均量变化 <1%。
- [ ] 净流方向、回吸比例和喷孔速度物理合理。
- [ ] 形成可被整机调用的传递函数/边界表。

## P4 整机气动

- [ ] 完整产品全部 cell 和实际进排气域均包含。
- [ ] 检查同相、反相/分组相位。
- [ ] 总质量守恒误差 <0.5%。
- [ ] 各 cell 流量分布和饥饿区域已解释。
- [ ] 至少一个无对称整机算例。
- [ ] 分开完成低背压非零流量/热输运与最高压力能力扫描；1750 Pa 只按产品压力能力检查，不虚构该点流量。
- [ ] 压力脉动/声学代理与 50 cm 系统噪声趋势建立可解释关系，但不把 CFD 压力脉动直接冒充 21 dBA 测量值。

## P5 整机 CHT

- [ ] 芯片热和 AirJet 自热分开统计。
- [ ] 总能量误差 <1%。
- [ ] 85 °C die、25 °C ambient 工况定义一致。
- [ ] 5.25 W 总耗散与 4.25 W 净移除定义一致。
- [ ] TIM、扩散板和环境边界的不确定性已报告。

## P6 标定与验证

- [ ] 至少三个独立指标参与标定。
- [ ] 至少一个公开指标保留为验证集。
- [ ] 主模型和可接受备选模型均保留。
- [ ] 无法唯一识别的内部参数明确列出。
- [ ] 所有结果可追溯至 CAD、参数表、求解器设置和 Git commit。
